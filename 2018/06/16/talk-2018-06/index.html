不知不觉又过去的一个月… 看了看上一次记录是在 2017年11月，已经过了半年了啊，所以也就写写这半年都干了些什么吧（x

去年10月我受同学邀请进入了 北京冰狮科技 工作也作为实习，这是一家从事手游开发、运营和提供平台对接的公司，我以 Java 开发的职位入职，不过说实话…这半年以来倒是没怎么写 Java 的代码，倒是写了一堆 Python 和 Javascript，还干了点运维的活…不过这里工资还是比较可观的…
因为6月初就是毕设答辩了，我之前还什么都没做，所以我于今年4月底我辞职离开了公司专心完成毕设的相关内容。

这半年依赖最重要的事当然就是毕业设计啦，虽说给了大半年的时间，但是…我5月份之前几乎就没怎么写…就写了一些通用的库，5月份肝了一个月的毕业设计和论文，终于在毕设查重前肝完了。
6月2日第一次答辩，因为功能太多了足足答了40多分钟…不过老师好像没什么想问的，大概是因为我用了一些近两年新出现的东西的缘故吧…
然后毕设被顺利推优，6月5日参加推优答辩，紧赶慢赶还是用了20分钟才答完，现在在等着19日校级答辩。
说实话，本科的毕业设计想通过真的非常容易，一个 RSS 爬虫加上一个关键词提取，加上数据存一下，然后关键词匹配推荐就能通过，大概也就几个小时的工作量。当然了，如果这么干的话要保证，论文会吹…然后在论文提交前一天我用4个小时给同学搓了一个出来…
如果想做成优秀毕设的话，那首先是要功能多、代码量大、论文字数和页数多（我的论文一共62页），还需要有一定的特色，以及解决一个实际的问题，当然这个实际的问题只要你能说清楚这个问题就行。
还有写毕设论文，一定要遵守格式要求，这个要求很严格，非常麻烦，我建议 先调好 Word 的样式，然后用样式正确标记各个部分，这样改起来会方便很多很多。

虽然做完了毕设，但是我还得上个大五，我在大三的时候从电子信息工程转转业到软件工程，导致学分不够，还得再多修一些学分才能毕业。所以下一学年应该会在没事的时候挖坑填坑什么的…

虽然写代码的能力没增强太多，但是电脑上用 Java 写的代码却越来越多了，而且大部分代码也几乎不会再去用了，所以我想着，整理一下，把有用的代码做成库开源，然后发布到一个 Maven 仓库去，这样也算以后需要用的时候方便，顺便还能偷个懒…
然后我看了一圈发现发布到各个 Maven 仓库的注册和配置都好麻烦…所以就自己简单的用 Github Pages 建了一个，反正库也不多，凑合用了。
现在可以访问 天空仓库 来访问这个 Maven 仓库。

这半年我的屋子也发生了很大的变化，因为电子设备太多了，还有一个比较大的屏幕，而屋里又没有一个合适的桌子，所以就自己画图去家具城定做了一个桌子。

中间的隔层用来放设备，现在里面塞着 PS4 Slim、XBox One S、千兆交换机、树莓派、HDMI 切换器、HDMI 复制器，以及各种适配器。然而两台游戏主机基本吃灰，买了几个游戏到现在只打通了看门狗2…古墓丽影9和10、怪物猎人世界、底特律、士官长合集都没怎么玩，然后 Kinect 运动竞技和舞力全开2018玩了几次也放那吃灰了…
现在也就有时候和 EdenChen 一起联机打打游戏什么的，我自己可能真的没什么动力玩游戏吧…（但看来收藏癖很严重啊！）

我之前一直是把我主力的笔记本的电脑分出了 200G 分区用来安装 GNU/Linux 系统，但这很不方便，每次切换很繁琐，所以就用自己的工资在闲鱼上入了一台 Thinkpad x220，换了一块电池，续航6个小时完全够用，这机器比较神奇，Intel 固态硬盘、8G 内存、千兆有线网卡，千兆带 5G 的无线网卡，但是它的 USB 口只有3个 2.0 的口，而且没有 3.0 控制器，我尝试在它的 ExpressCard 插槽中插入了一个 ExpressCard to USB 3.0 卡，但是插上去可以认卡，一旦插入任何 USB 设备就会整个卡离线，所以也就没法加 3.0 插口了。
拿到手之后测试了没有问题，然后格式化电脑安装了 openSuSE Tumbleweed 系统，选用 KDE5 Plasma 作为桌面。经历了2次 btrfs 崩溃后，更换至 ext4 安稳使用。

最近看到我在 B 站客户端上追过的番已经有100多部了，但是 B 站有很多很多很多番剧都没有，比如 未来日记 啊、 未来日记 啊、 未来日记 啊。我妻由乃真是太棒了！也有很多番剧有删减和圣光暗牧，所以也想统计下我自己看过的番，和一直想补的番，以防我自己忘记，所以花了一点时间整理了一下 天空看的番 ，现在在博客侧边栏上可以找到这个链接，不过这里面缺少很多以前看过的动画，特别是欧美的动画，我会尽可能地补上。
