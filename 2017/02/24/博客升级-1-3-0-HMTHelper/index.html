经过了1个月的难产… hexo-material-theme 迎来了一个大版本升级 1.3.0 。在这次的升级中修复了古老的代码染色 bug ，开发了新插件用于支持博文置顶，增加了 Disqus 的评论点击加载（访客模式绝赞开发中！），全自定义侧边栏，剩下的就是一些小修补。同时，我们引入了加载队列来防止 js 文件在页面未完成渲染之前集中加载造成某些关键功能使用不流畅。
而本博客也在前天（2017-02-22）升级到了新版本。

本来写的昨天，一看表已经过零点了…..

同时，首页的一言横幅也被我重构，现在使用我开发的 Hexo-Material Head Helper JS 进行加载控制，不再在所有页面加载整个文件后再判断，大幅度降低相关文件读取时间。
