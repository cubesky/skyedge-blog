在非 Windows Server 系统上启用 RemoteApp Host

没错！我就是来水博客的！（理直气壮）
之前我在我的 Telegram 频道 里说过用 RemoteApp 来进行类似本地软件体验的远程访问的方法。但那个方法要架设 Windows Server 服务器，激活 RemoteApp 服务并且要搞各种许可证，非常的麻烦。
现在！我们有一个全新的方法啦！不需要3个小时，只需要3分钟（只要在支持列表里）就可以体验到 RemoteApp！
我们只需要一个名为 RemoteApp Tool 的开源软件，打开它的 Github 页面并在下方 Download 中下载 Latest Installer，不要忘记在 Requirements 里面下载 WiX Toolset。安装好之后，在开始菜单找到 RemoteApp Tool, 打开它！

注：要查看您的系统是否支持，访问 https://github.com/kimmknight/remoteapptool/wiki/Windows-Compatibility 上面写着 YES 的就可以。

之后就很简单了，点击左下角的加号，选择程序，这样就完成了添加操作。然后选中这个程序点击右下角的 Create Client Connection... 就会弹出这个界面。

在最上面选择 RDP，然后把 Remote Server 中的 Server address 改成你这台电脑可访问的地址就可以了。点击右下角 Create 创建 RDP 文件。

之后你只需要在其他电脑上双击这个 RDP 并连接就可以啦！

如果你想创建 MSI 安装程序就选择 MSI installer 具体内容可以在下方自己研究w
